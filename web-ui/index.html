Queue Service for your DApps

ICSQS is a native, scalable decentralized simple message queuing service powered by the internet computer.

IC SQS Offers

Reliable message queue

Create a reliable message queue that can be used to DApps to IOT devices as queue service.

Batch Operations

IC SQS offers batch operations, so you can push or delete multiple messages to a queue at once.

Visibility

Message visibility timeout is the amount of time that a message is invisible in the queue after a reader picks up the message. IC SQS makes it easy to customize the visibility timeout for your queue.

Security

Only authorized identities are allowed to access the queue. These identities can be updated according to the requirement.

Fully open source DApp

you can use the source code to build & deploy your own SQS service. For build & deploy, easy instructions are available on our github page.

Internet Computer

IC SQS is build on top of the internet computer, a decentralized, open-source, and secure network.
